# Monitoraggio Sicurezza Server

Questa sezione documenta le attività di monitoraggio e analisi del traffico in ingresso verso il server, focalizzandosi sull'identificazione di tentativi di accesso non autorizzati.

## Obiettivo
Mantenere la visibilità sulle minacce attive e validare l'efficacia delle policy di hardening applicate.

## Attività
- **Monitoraggio log di sistema**: Analisi continua del file `/var/log/auth.log`.
- **Analisi delle minacce**: Identificazione di pattern di attacco (brute force, scansioni di rete).

## Report disponibili
- [Analisi Log SSH](ssh-log-analysis.md): Report dettagliato sugli attacchi rilevati e sulle misure di protezione attive.
